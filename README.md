# Projeto-E-Commerce
Análise de Vendas de um E-commerce

Esse projeto explora e analisa dados reais de um E-Commerce brasileiro (Olist) para entender o comportamento dos clientes e gerar insights que apoiem decisões estratégicas.

# Ferramentas utilizadas
Python - Pandas e Numpy para limpeza e manipulação dos dados.
Seaborn e Matplotlib - criação de gráficos e visualizações.
Github - documentação e apresentação do projeto como portifólio.

# Etapas realizadas
- Coleta e carregamento dos dados -> importação dos arquivos CSV da Olist na Kaggle.
- Limpeza e integração -> tratamento de nulos, duplicados e junção de tabelas (clientes, pedidos e itens).

# Análises principais
- Ticket médio -> valor médio gasto por pedido.
- Produtos mais vendidos -> identificação dos itens mais populares.
- Vendas por estado -> comparação regional do volume de vendas.

# Visualização 
- Mini-dashboard com gráficos estilizados para comunicar os resultados.

# Resultados esperados
- Identificação de padrão de consumo.
- Insights sobre regiões mais lucrativas.
- Métricas claras para apoiar estratégias de marketing e vendas.
